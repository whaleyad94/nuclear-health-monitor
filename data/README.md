# Data

The dataset used in this project is the **Combined Cycle Power Plant (CCPP) Dataset**
from the UCI Machine Learning Repository.

## Download Instructions

1. Download the dataset from:
   https://archive.ics.uci.edu/ml/machine-learning-databases/00294/CCPP.zip

2. Unzip and place `Folds5x2_pp.xlsx` in this `/data` directory.

## Dataset Description

9,568 hourly observations collected from a Combined Cycle Power Plant over 6 years.
